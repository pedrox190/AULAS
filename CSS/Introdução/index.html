O que é CSS

CSS CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo utilizada para descrever a apresentação d edocumentos escritos em HTML ou XML. Ela define a aparencia visual de uma pagina web, controlando elementos como cores, fontes, espaçamentos, layout e design.

Pagina inicial Sobre

Existem 3 maneira de aplicar CSS

Inline

Na mesma linha, atraves de um atributo chamado style

Interno

Dentro do mesmo documento do HTML, atraves de uma tag chamada style

Externo

Criando um novo arquivo .css e utilizando seletores
